CSS란

HTML 이 웹 문서의 구조를 담당한다면
CSS 글꼴 , 색상 , 여백 같은 스타일을 추가해야함
보기좋게 꾸며주는 역할을 담당한다.

적용방법

  • 내부 스타일 , style 태그
  • 외부 스타일

선택자

  • 전체선택자 *
  • 태그선택자 tag
  • 클래스 선택자 . : 중복허용
  • 아이디 선택자 # : 한 개
  • 자손 선택자 공백 (스페이스) : li.orange
  • 자식 선택자 >
  • 인접형제 선택자 +
  • 인접형제 선택자 ~
  • 일치 선택자 li.orange => li 태그이면서 orange 클래스를 가진 요소 선택
  • 가상 클래스 선택자 :hover :active :focus
  • 가상 요소 선택자 ::befor , ::after
