데이터타입과 변수

  • 숫자타입 : 42, 42.7
  • 논리타입 : true, false
  • 문자열타입 : "hello" , "apple" , "a"
  • 객채타입

문자, 문자열 , 자바스크립트는 문자열로만 취급함

"hello" => 문자의 집합 (문자열) "h" => 한 문자 (자바스크립트는 문자없음)

변수 : 값을 저장하기위한 메모리 블럭 (기억장소)

변하는 수를 의미함 let kor=90;
kor은 변수, 변할 수 있음
kor=95; kor은 이제 95의 값을 유지함

변수의 역할 : 기억, 이용

값은 이용하기 위해 저장된다.
